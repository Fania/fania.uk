Workflow

    install python (2.7) install pip (To enable the use of pip from the command line, ensure the Scripts subdirectory of your Python installation is available on the system PATH. (This is not done automatically.) make a virtualenv pip install -r requirements.txt

Virtual Env & Pip

  • Create a new venv: virtualenv name
  • Activate . name/bin/activate
  • Activate name\scripts\activate
  • Deactivate deactivate
  • To freeze reqs pip freeze --local > requirements.txt
  • To freeze reqs pip freeze > requirements.txt
  • To install from reqs pip install -r requirements.txt
  • virtualenv --no-site-packages --distribute .env && source .env/bin/activate && pip install -r requirements.txt

Open terminal or cmd at location

  • 'Shift-right-click' the folder to open cmd there
  • Use the 'Open-In-Terminal' script
